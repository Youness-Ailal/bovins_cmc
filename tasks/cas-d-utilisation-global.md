# Cas d'utilisation global

## Administrateur

```text
- Se connecter
- Voir son profil
- Gerer les utilisateurs
- Creer un utilisateur
- Modifier un utilisateur
- Changer le mot de passe d'un utilisateur
- Modifier le role d'un utilisateur
- Activer ou desactiver un utilisateur
- Gerer les races
- Gerer les parametres de la ferme
- Configurer les alertes
- Acceder a toutes les pages
```

## Responsable elevage

```text
- Consulter le dashboard
- Gerer les animaux
- Creer un animal
- Modifier un animal
- Consulter la fiche animal
- Ajouter une pesee
- Changer la phase d'un animal
- Declarer une sortie: vente, abattage ou mortalite
- Generer passeport, laissez-passer et QR code
- Voir les animaux prets a vendre
- Gerer les parcelles
- Creer une parcelle
- Modifier une parcelle
- Transferer un animal entre parcelles
```

## Responsable stock et alimentation

```text
- Supprimer un article

- Consulter l'historique des mouvements
- Voir les seuils et alertes de stock

- Modifier une ration

- Distribuer une ration
- Consulter l'historique des distributions
```
